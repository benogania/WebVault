# WebVault: The  Offline Browser & Site Manager

WebVault is a Android application designed for users who need reliable access to web content without an internet connection. It allows you to organize your favorite sites and download pages for true offline viewing.

## 🚀 Key Features

* **Smart Offline Bundling**: Unlike standard "Save Page" features, WebVault fetches and inlines CSS directly into a single file, ensuring the layout and styling remain perfect even offline.
* **AI Site Recommendations**: Discover new tools and resources with a built-in recommendation engine that suggests useful sites to add to your vault.
* **Integrated Web Viewer**: Browse live sites directly in the app and save them to your vault with a single tap.

* **Advanced Management**: Organize links into folders, pin important resources, and manage your collection with intuitive long-press actions (Edit/Delete).
* **Visual Dashboard**: Automatically pulls high-quality website icons (favicons) to give your vault a professional, clean look.

## 🛠️ Tech Stack

* **Framework**: React Native (Expo)
* **Styling**: NativeWind (Tailwind CSS)
* **Storage**: AsyncStorage & Expo FileSystem
* **Components**: React Navigation, Expo WebView, Ionicons

## 🛠️ Local Development

To run this project locally:

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/benogania/WebVault.git](https://github.com/benogania/WebVault.git)

    cd WebVault
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the project**:
    ```bash
    npx expo start
    ```
